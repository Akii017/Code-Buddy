# Code Buddy Chrome Extension

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the extension:
   ```bash
   npm run build
   ```
3. Load the `public/` folder as an unpacked extension in Chrome.

## Features

- Sleek popup UI (React + Shadcn UI)
- Content script for LeetCode/other platforms
- AI-powered hints and explanations (backend integration coming soon)
